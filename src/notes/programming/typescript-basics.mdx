## TypeScript and React
### Props
- Typically define props for a component like so:
```
// Define interface
interface AppProps {
  props: data-type
}

// Define props type
const App = ({ props }: AppProps) {
  return (
    ...
  )
}
```

- For optional props, use `?`
- You can also set default props
